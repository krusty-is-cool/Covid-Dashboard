# Covid-Dashboard
A simple dashboard

It is using Flask microframework.

Client :
- HTML
- CSS
- JavaScript

Server:
- Python

Not hosted at the moment.

## Features

Select a country, and get a nice graph of the daily new Covid-19 cases in this country. Moving average on 7 days to filter the data.
